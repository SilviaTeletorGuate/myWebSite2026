# Portafolio Web de Silvia

¡Hola! Soy **Silvia**
Este proyecto es mi portafolio personal, donde muestro mis habilidades en diseño y desarrollo web, experiencia de usuario (UX/UI) y proyectos formativos.

##  Descripción del Proyecto

Esta página web está diseñada para presentar mi perfil profesional y mis experiencias en desarrollo web. Contiene secciones interactivas para:

- **Inicio:** Presentación personal y breve descripción.
- **Acerca de mí:** Información sobre mi enfoque en diseño centrado en el usuario y buenas prácticas de desarrollo.
- **Proyectos / Experiencia:** Experiencias formativas y trabajos realizados, destacando habilidades técnicas y aprendizaje continuo.
- **Contacto:** Formulario para que visitantes puedan enviarme mensajes directamente desde la web.

El proyecto está construido con **HTML**, **CSS** y **JavaScript**, y está pensado para ser **responsivo**, visualmente atractivo y fácil de navegar.

##  Tecnologías Utilizadas

- **HTML5** – Estructura del sitio web.
- **CSS3** – Estilos y diseño responsivo.
- **JavaScript** – Interactividad y navegación dinámica.
- Imágenes y recursos gráficos optimizados para web.
